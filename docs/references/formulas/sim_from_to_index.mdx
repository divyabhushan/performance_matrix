## Syntax

`SIM_FROM_INDEX` starting periond index of simulation range selected in navigation pannel
`SIM_TO_INDEX` ending periond index of simulation range selected in navigation pannel

## Example
If Simulation period is seletected as Feb to June then `SIM_FROM_INDEX` will be 2 and `SIM_TO_INDEX` will be 6
```javascript
SIM_TO_INDEX - SIM_FROM_INDEX // will return 4
```
